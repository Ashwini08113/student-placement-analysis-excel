# 📊 Placement Success Analysis Dashboard

## 📌 Project Overview

The **Placement Success Analysis Dashboard** is an Excel-based data analytics project designed to analyze student placement trends and identify the factors influencing job placements.

This dashboard provides insights into placement performance using various parameters such as **college tier, internships, skill level, CGPA category, and gender-based salary comparison**.

The project uses **data analysis techniques, pivot tables, charts, and interactive slicers** to build a dynamic dashboard for better decision-making.

---

## 🎯 Project Objectives

* Analyze overall student placement performance.
* Identify how **college tier impacts placement outcomes**.
* Understand the influence of **internships on student placements**.
* Compare placements based on **skill levels**.
* Study placement distribution across **CGPA categories**.
* Analyze **average salary differences by gender**.
* Create an **interactive dashboard** for visual data analysis.

---

The dashboard highlights important placement indicators:

* **Average Salary Package:** 13.32 LPA

These KPIs help quickly understand the placement performance.

## 📈 Dashboard Visualizations

The dashboard includes multiple charts to analyze placement trends:

### 1️⃣ Placement Distribution by College Tier

Compares the number of placed students across **Tier 1, Tier 2, and Tier 3 colleges**.

### 2️⃣ Impact of Internships on Student Placements

Shows how **internship experience affects placement chances**.

### 3️⃣ Placement Count by Skill Level

Analyzes placement outcomes for students with **High, Medium, and Low skill levels**.

### 4️⃣ Placement Distribution by CGPA Category

Displays placement distribution among:

* Highly Employable
* Moderately Employable
* Needs Improvement

### 5️⃣ Average Salary Comparison by Gender

Compares the **average salary package offered to male and female students**.

---

## 🎛 Interactive Dashboard Features

The dashboard includes **slicers (filters)** that allow users to interact with the data:

* **Skill Category Filter**
* **College Tier Filter**
* **CGPA Category Filter**

These slicers dynamically update charts and help users explore placement trends more effectively.

---

## 🔍 Key Insights

Some important insights derived from the dashboard:

* **Tier 2 colleges produced the highest number of placements.**
* Students with **internship experience have higher placement success.**
* **Skill level plays a significant role** in determining placement outcomes.
* Students categorized as **Highly Employable show strong placement performance.**
* The dataset shows **minor salary variation between genders**.
