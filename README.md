
# 🎓 Student Placement Analysis Dashboard

!["C:\Users\Ashwi\OneDrive\Pictures\dashboard screenshot.png"](screenshot%20dashboard.png)

## 📌 Project Overview

This project analyzes **student placement data** using **Microsoft Excel** to uncover trends, placement rates, and hiring patterns.

The goal of this project is to transform raw student data into **interactive insights using Pivot Tables, Charts, and a dynamic Excel dashboard**.

This project demonstrates **data analysis, data visualization, and dashboard development skills**, which are essential for **Data Analyst roles**.

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Dashboard Design

---

## 📂 Dataset Information

The dataset contains student placement related attributes such as:

* Student ID
* College Tier
* CGPA Category
* Skill Category
* Internship Count
* Placement Status
* Salary Package (LPA)
* Gender
* Lifestyle Balance
* Study Hours
* Extracurricular Score

The data was analyzed to understand **patterns affecting student placements**.

---

## 📊 Dashboard Features

The Excel dashboard provides the following insights:

* 📈 Total Students vs Placed Students
* 🎯 Placement Rate Analysis
* 🏫 Department-wise Placement Statistics
* 💰 Salary Distribution
* 🏢 Company-wise Hiring Trends

Interactive **Slicers** allow users to dynamically filter and explore the data.

---
## 📈 Dashboard Visualizations

The dashboard includes multiple charts to analyze placement trends:
* 1️⃣ Placement Distribution by College Tier
Compares the number of placed students across Tier 1, Tier 2, and Tier 3 colleges.
* 2️⃣ Impact of Internships on Student Placements
Shows how internship experience affects placement chances.
* 3️⃣ Placement Count by Skill Level
Analyzes placement outcomes for students with High, Medium, and Low skill levels.
* 4️⃣ Placement Distribution by CGPA Category
Displays placement distribution among:

Highly Employable
Moderately Employable
Needs Improvement
* 5️⃣ Average Salary Comparison by Gender
Compares the average salary package offered to male and female students.
